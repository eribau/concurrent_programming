To build and run the programs, execute the run shell script inside their 
respective folder.

Example:
/hw1/matrixSum/$ ./run.sh

When running the pi-program, it will be run with the parameters of 4 2 0.000001
where the 1st parameter is #of threads, 2nd parameter is a multiplier to set
the max number of generated tasks and the 3rd paramter is the epsilon value.
